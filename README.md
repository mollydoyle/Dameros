## Installation
1. Clone repository
```
git clone https://github.com/tclare/Dameros.git
```
2. Create virtual environment
```
python3 -m venv venv
```
3. Activate virtual environment
```
source venv/bin/activate
```
4. Install requirements
```
pip3 install -r requirements.txt
```
5.  Run server
```
flask run
```
